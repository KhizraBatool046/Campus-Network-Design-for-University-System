#  Campus Network Design for a University System

## Project Overview

This project was developed as part of the **Computer Networks** course during the fifth semester of my BS Computer Science degree.

The project focuses on designing and simulating a university-wide Campus Area Network (CAN) using Cisco Packet Tracer. Multiple campuses and departments are connected through routers, switches, LANs, and WAN links to provide reliable communication and resource sharing.

---

## Features

- University campus network design
- LAN and WAN implementation
- Static routing configuration
- IP addressing and subnetting
- Router and switch configuration
- End-to-end connectivity testing
- Network troubleshooting
